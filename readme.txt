Návod na spuštění

Pro spuštění je třeba mít nainstalován v počítači Python (verze 3.11.9).
Pomocí příkazu python3 -m venv .env je nutné vytvořit si virtuální prostředí.
Nainstalovat balicky z vyse uvedeneho source .env/bin/activate && pip install -r requirements.txt, které jsou v repozitáři.
Potí zadat příkaz flask --run.
Aplikace běží na adrese http://127.0.0.1:5000
